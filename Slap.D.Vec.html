val sort : ?cmp:(Slap.D.num_type -> Slap.D.num_type -> int) ->
       ?decr:bool ->
       ?p:('n, 'p_cd) Slap.Common.int_vec -> ('n, 'x_cd) Slap.D.vec -> unit
sort ?cmp ?decr ?p x sorts the elements in vector x in increasing order according to the comparison function cmp.
cmp : default = the usual order (on real numbers) or the lexicographic order (on complex numbers).
  • cmp a b < 0 iff a < b;
  • cmp a b = 0 iff a = b;
  • cmp a b > 0 iff a > b.
  • NaNs must be considered larger than any other value.
decr : sort in decreasing order.
p : If p is passed, x is unchanged and the permutation is stored into p. (default = None.)

val subcntvec_dyn : 'm Slap.Size.t ->
       ?ofsx:int -> ('n, Slap.cnt) Slap.D.vec -> ('m, 'cnt) Slap.D.vec
subcntvec_dyn m ?ofsx x
Raises Invalid_argument m and ofsx do not designate a valid subvector of x.
