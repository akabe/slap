Creation of matrices


val create : ('num, 'prec) Bigarray.kind ->
       'm Common.size -> 'n Common.size -> ('m, 'n, 'num, 'prec, 'cnt) Common.mat
create kind m n
Returns a fresh m-by-n matrix (not initialized).
val make : ('num, 'prec) Bigarray.kind ->
       'm Common.size ->
       'n Common.size -> 'num -> ('m, 'n, 'num, 'prec, 'cnt) Common.mat
make kind m n x
Returns a fresh m-by-n matrix initialized with x.
val init_cols : ('num, 'prec) Bigarray.kind ->
       'm Common.size ->
       'n Common.size ->
       f:(int -> int -> 'num) -> ('m, 'n, 'num, 'prec, 'cnt) Common.mat
init_cols kind m n ~f returns a fresh m-by-n matrix whose the (i,j) element is initialized by the result of calling f i j. The elements are passed by column-major order.
val init_rows : ('num, 'prec) Bigarray.kind ->
       'm Common.size ->
       'n Common.size ->
       f:(int -> int -> 'num) -> ('m, 'n, 'num, 'prec, 'cnt) Common.mat
init_rows kind m n ~f returns a fresh m-by-n matrix whose the (i,j) element is initialized by the result of calling f i j. The elements are passed by row-major order.

Basic operations


val fill : ('m, 'n, 'num, 'prec, 'cd) Common.mat -> 'num -> unit
Fill the given matrix with the given value.
val copy : ?b:('m, 'n, 'num, 'prec, 'b_cd) Common.mat ->
       ('m, 'n, 'num, 'prec, 'a_cd) Common.mat ->
       ('m, 'n, 'num, 'prec, 'b_cd) Common.mat
copy ?b a copies the matrix a into the matrix b.
Returns b, which is overwritten.
b : default = a fresh matrix.

replace_all a ~f modifies the matrix a in place -- the (i,j)-element aij of a will be set to f aij.
val replace_alli : ('m, 'n, 'num, 'prec, 'cd) Common.mat ->
       f:(int -> int -> 'num -> 'num) -> unit
replace_all a ~f modifies the matrix a in place -- the (i,j)-element aij of a will be set to f i j aij.

Submatrices


val submat_dyn : 'm Common.size ->
       'n Common.size ->
       ?ar:int ->
       ?ac:int ->
       ('a, 'b, 'num, 'prec, 'cd) Common.mat ->
       ('m, 'n, 'num, 'prec, Common.dsc) Common.mat
submat_dyn m n ?ar ?ac a
Returns a m-by-n submatrix of the matrix a. The (i,j) element of the returned matrix refers the (ar+i-1,ac+j-1) element of a. The data are shared.
ar : default = 1
ac : default = 1
