Module Slap.Size

module Size: sig .. end
Sizes (dimensions of vectors and matrices).
type 'n t = private int
A singleton type on sizes (i.e., dimensions of vectors and matrices).

Evaluation of a term with singleton type 'n Size.t always results in the natural number corresponding to phantom type parameter 'n. 'n is instantiated to a generative phantom type or a (phantom) type that represents an arithmetic operation defined in this module. In either case, only the equality of sizes is verified statically.
